# Mobile NexoCore

Esta carpeta queda reservada para el futuro desarrollo mobile de NexoCore.

## Estado actual

La aplicación mobile todavía no forma parte activa del proyecto.

Actualmente el sistema principal está compuesto por:

* Backend API
* Frontend web
* Base de datos PostgreSQL
* Documentación Swagger

## Posible stack futuro

Cuando se inicie el desarrollo mobile, se recomienda evaluar:

* React Native
* Expo
* Axios
* React Navigation
* Integración con la API existente de NexoCore

## API base

La app mobile deberá consumir la misma API del backend:

```txt
http://localhost:4000/api
```

En producción, deberá configurarse la URL pública del backend.

## Módulos futuros sugeridos

* Login mobile
* Dashboard resumido
* Operaciones asignadas
* Carga de evidencias desde cámara o galería
* Notificaciones
* Consulta de estado de operaciones

## Nota

Esta carpeta se mantiene preparada para una etapa futura del proyecto.
